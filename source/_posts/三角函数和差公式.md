公式：

```
sin(δ+β) = sin(δ)cos(β) + cos(δ)sin(β)
cos(δ+β) = cos(δ)cos(β) - sin(δ)sin(β)
```

所以得出：

```
c = r*cos(δ+β) = r*cos(δ)cos(β) - r*sin(δ)sin(β) = x * cos(β) - y * sin(β)
d = r*sin(δ+β) = r*sin(δ)cos(β) + r*cos(δ)sin(β) = y * cos(β) + x * sin(β)
```

需要的最终结果：

```
c = x * cos(β) - y * sin(β)
d = y * cos(β) + x * sin(β)
```

此结果用于旋转象限非常适用。

像游戏`2048`就能用到。<br>
